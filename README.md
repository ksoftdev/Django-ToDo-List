# Django ToDo List

This is a pratical ToDo application using django framework.

Covers:

* Django class based views
* User login
* User registration
* Create, Delete and Modify tasks list
* Search functionality

Technologies:

* Django (Backend)
* HTML, JavaScript, CSS (Frontend)
